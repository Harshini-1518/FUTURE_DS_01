# FUTURE_DS_01 – Business Sales Performance Analytics

# Overview :
This project was completed as part of the Future Interns Data Science & Analytics Internship Program.
The objective of this analysis is to evaluate business sales data and identify key revenue trends, top-performing products, category performance, regional sales distribution, and monthly revenue patterns.


# Tools Used : 
* Python
* Pandas
* Matplotlib
* Microsoft Excel


# Files Included :
* `FUTURE_DS_01.xlsx` – Sales dataset used for analysis
* `FUTURE_DS_01.py` – Python script for data cleaning, KPI calculation, and visualization
* `README.md` – Project documentation


# Analysis Performed :

## 1. Data Cleaning : 
* Loaded sales data from Excel
* Removed missing values
* Prepared data for analysis

## 2. KPI Analysis :
Calculated:
* Total Revenue
* Average Revenue
* Total Orders

## 3. Top-Selling Products :
* Identified the top 10 products based on revenue generated
* Created bar chart visualizations

## 4. Category Performance Analysis :
* Analyzed revenue contribution by product category
* Compared category-wise sales performance

## 5. Regional Performance Analysis :
* Evaluated sales performance across different regions
* Identified high-performing regions

## 6. Monthly Revenue Trend Analysis :
* Converted order dates into monthly periods
* Generated monthly revenue trends
* Visualized sales growth patterns over time

## Key Insights :
* Identified products generating the highest revenue.
* Determined the best-performing product categories.
* Compared sales performance across regions.
* Analyzed monthly trends to understand business growth.

## Visualizations :
The project includes:
* Top 10 Selling Products Chart
* Category-wise Sales Chart
* Region-wise Sales Chart
* Monthly Revenue Trend Chart

## Outcome :
This analysis provides actionable business insights that can help organizations improve sales strategies, optimize product offerings, and make data-driven decisions.
